# 🎣 Phishing-FIAP-Emerson

## 📚 Sobre o Projeto
Este projeto foi desenvolvido como uma atividade acadêmica com o objetivo de **estudar e compreender como ataques de phishing funcionam na prática**.

A proposta não é incentivar práticas maliciosas, mas sim **aprender a identificar vulnerabilidades** e entender quais erros devem ser evitados ao desenvolver sistemas web.

## 🎯 Objetivo
- Simular uma página de login (phishing)
- Entender como usuários podem ser enganados
- Identificar falhas comuns de segurança
- Aprender boas práticas para desenvolvimento seguro

## 🛠️ Tecnologias Utilizadas
- HTML
- CSS
- PHP
- SQL

## 📂 Estrutura do Projeto
- `FIAP EAD - Login.html` → Página simulando login
- `login.php` → Processamento dos dados
- `bancoFiapFishing.sql` → Estrutura do banco de dados
- `FIAP EAD - Login_files` → Arquivos auxiliares (CSS, imagens, etc.)

## 🔍 O que foi aprendido
- Como páginas falsas imitam sistemas reais
- Importância de validação de dados
- Riscos de engenharia social
- Necessidade de HTTPS e proteção de credenciais
